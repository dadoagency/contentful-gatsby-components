## How to install

`yarn add @dadoagency/contentful-gatsby-components`

### Edit gatsby config

```
module.exports = {
  ...
  plugins: [
        `@dadoagency/contentful-gatsby-components`,
  ]
}
```

## How to develop

```
yarn
yarn link
```
